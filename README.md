`leetcode 300`
最长递增子序列的长度，，，简单做法就是上面 `max(dp[i])`。更优化的方法是贪心 + 二分，构建单调序列 `d[i]` 代表长度为 i 的序列的最小结尾值。
如果比当前序列结尾值大，插入到结尾，否则通过二分查找第一个比 `d[i]` 小的 `d[k]` 然后更新最小值 `d[k+1] = d[i]`。最后看递增序列的长度。
贪心体现在让单调递增序列中值尽可能的小。
